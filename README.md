# Aprendiendo a hacer ETL: Formación en Ciencia de Datos

Este repositorio contiene los ejercicios, proyectos y apuntes desarrollados durante mi formación en el procesamiento y manipulación de datos. El enfoque principal está puesto en las etapas de **Transformación (T)** y **Carga (L)** de un pipeline ETL, utilizando el ecosistema de Python para asegurar la eficiencia en el análisis numérico y la gestión de grandes volúmenes de datos.

## Estructura del Contenido

El aprendizaje se divide en cuatro pilares fundamentales que recorren desde la manipulación de matrices hasta la implementación de Inteligencia Artificial para la visualización:

### 1. Manipulación de Datos con NumPy
Se incluyen ejercicios sobre el uso de la biblioteca NumPy para el análisis numérico eficiente.
* Operaciones matemáticas y estadísticas sobre matrices.
* Indexación y selección avanzada de datos.
* Optimización de cómputo científico.

### 2. Procesamiento y Transformación con Pandas
Módulos dedicados a la biblioteca Pandas, herramienta central para la manipulación de estructuras de datos tabulares.
* **Pandas E/S:** Carga y lectura de datos en múltiples formatos (CSV, JSON, Excel, etc.).
* **Transformación:** Técnicas de limpieza, normalización y preparación de DataFrames.
* **Selección y Agrupamiento:** Métodos avanzados de filtrado y agregación de información para análisis específicos.

### 3. Visualización de Datos
Prácticas enfocadas en la representación gráfica de la información para facilitar la toma de decisiones.
* **Matplotlib:** Creación de gráficos de líneas, barras y dispersión.
* **Estilización:** Personalización de gráficos y tablas para una comunicación efectiva.

### 4. IA Aplicada al Análisis de Datos
Implementación de herramientas de Inteligencia Artificial para optimizar el flujo de trabajo.
* Uso de ChatGPT como asistente para la generación de código de análisis.
* Creación de informes interactivos mediante Gamma y herramientas de visualización asistida por IA.

## Tecnologías Utilizadas

| Categoría | Herramientas |
| :--- | :--- |
| **Lenguaje** | Python |
| **Librerías de Datos** | NumPy, Pandas |
| **Visualización** | Matplotlib, Seaborn |
| **Asistentes IA** | ChatGPT, Gamma |

## Objetivo del Repositorio
El código guardado acá sirve como bitácora de aprendizaje y referencia técnica para futuros proyectos de ingeniería de datos, asegurando que los procesos de limpieza y transformación sigan las mejores prácticas de la industria.
