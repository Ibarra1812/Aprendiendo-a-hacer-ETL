# Aprendiendo-a-hacer-ETL
Aprendiendo a hacer ETL: Formación en Ciencia de Datos Este repositorio contiene los ejercicios, proyectos y apuntes desarrollados durante mi formación en el procesamiento y manipulación de datos. El enfoque principal está puesto en las etapas de Transformación (T) y Carga (L) de un pipeline ETL.
